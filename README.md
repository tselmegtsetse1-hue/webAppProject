# Sneaker Store Web Application

Энэ нь KREAM, Footlocker-оос санаа авсан онлайн sneaker дэлгүүрийн веб төсөл юм.
Төсөл нь frontend (HTML, CSS, JavaScript) болон backend логик, Supabase database ашигласан.

## Ашигласан технологи
- HTML / CSS
- JavaScript
- Supabase (Database, Auth)

## Боломжууд
- Бараа жагсаалт харах
- Бараа нэмэх / устгах
- Хэрэглэгчийн мэдээлэл хадгалах
- Backend API ашиглан өгөгдөл удирдах

Энэхүү төсөл нь web application хичээлийн хүрээнд хийгдсэн.
